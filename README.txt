Copa Draft 2026 — 3 arquivos

Arquivos:
1. index.html — tela, visual e estrutura.
2. worldcup-data.js — base de dados com seleções e jogadores.
3. game-engine.js — regras, draft, simulação, rankings e estatísticas.

Para usar no GitHub Pages:
- Suba os 3 arquivos na raiz do repositório.
- O arquivo precisa se chamar index.html.
- Ative Pages em Settings > Pages > Deploy from branch.

Observação:
A base é jogável/provisória para o jogo, não escalação oficial fixa.
